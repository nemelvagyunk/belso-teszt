# Belső oldal – teszt

Beléptetős weboldal-teszt (usernév + jelszó, kliensoldali SHA-256 ellenőrzés).
Egy későbbi, rendes belső oldal előtanulmánya — érzékeny adat nincs és ne is
legyen benne, a kliensoldali kapu nem valódi biztonsági védelem.

Oldal: https://nemelvagyunk.github.io/belso-teszt/
Teszt-fiókok: `attila` / `teszt123` · `vendeg` / `vendeg`
